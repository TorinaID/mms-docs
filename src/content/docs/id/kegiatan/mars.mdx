### Tabel Laporan Mars

Halaman MARS terdiri dari daftar (_list_) kegiatan berdasarkan periode laporan yang sudah ditentukan yaitu periode Jumat - Kamis. terdiri dari kolom-kolom :
- **Periode**: Menampilkan data periode pengisian data yang diatur setiap hari **Jumat** sampai dengan **Kamis**
- **Radio**: Data pemutaran Mars BNN melalui siaran radio
- **Sosialisasi**: Data Sosialisasi yang dilakukan di Sekolah sampai dengan Universitas, antara lain:
  - TK,
  - SD,
  - SMP,
  - SMA,
  - Universitas
- **Kegiatan**: Data kegiatan lainnya
- **Total**: jumlah keseluruhan dari kegiatan dalam 1 periode
- **Terakhir Update**: Data tanggal pengisian data kegiatan
- **Keterangan**: menginformasikan pada periode tersebut terdapat kegiatan atau tidak ada kegiatan

## Pengisian Data Mars
---
### Ketentuan Pengisian Data
> Ketentuan Data ini diharapkan untuk dapat membantu dan memudahkan wilayah untuk membuatkan laporan dan agar memudahkan Admin untuk me-_monitor_ kegiatan

:::caution
1. Seluruh data dibagi berdasarkan hari dalam periode, yaitu data mulai __hari Jumat__, sampai dengan data __hari kamis__
2. __Tanggal kegiatan__ adalah tanggal dimana kegiatan dilaksanakan, sedangkan __Tanggal Update__ adalah tanggal pengisian kegiatan ke dalam aplikasi
3. __Penambahan__ data __Periode Baru__ hanya dapat dilakukan jika periode sebelumnya sudah di konfirmasi __terisi dengan data kegiatan__ atau __terkonfirmasi bahwa tidak ada kegiatan__.
   > Pada proses ini data Periode sebelumnya sudah tidak dapat diubah ataupun dihapus
4. __Ubah__ data Periode __hanya bisa__ dilakukan pada data __periode yang sedang berjalan__ atau data __periode terakhir__.
:::
